= Subjects App

This is an application powered by kmaps_engine[http://github.com/shanti-uva/kmaps_engine]
using the shanti_integration[http://github.com/shanti-uva/shanti_integration], developed
under the Sciences, Humanities and Arts Network of Technological Initiatives (Shanti) at
the University of Virginia.

This app is just a shell to integrate the various gems that contain the actual code.
